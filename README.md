## 📚 Content Description

### Cisco Certified Network Associate (CCNA)

The CCNA section contains materials from the Cisco Networking Academy curriculum, including:

- **Certifications**: Official completion certificates for:
  - Introduction to Networks
  - Switching, Routing, and Wireless Essentials
- **Final Project**: A comprehensive network design project with documentation
- **Packet Tracer Labs**: Practice scenarios covering:
  - DHCP Snooping, DAI, and Port Security
  - EtherChannel configuration
  - Inter-VLAN routing and static routes
  - HSRP, FTP, HDLC, and PPP
  - Access Control Lists (ACL), AAA, and DHCP
  - Syslog, NAT, CDP, and troubleshooting
  - OSPFv2 routing protocol

### Cisco Certified Network Professional (CCNP)

Advanced networking topics building on CCNA knowledge:

- **Packet Tracer Labs**: Complex scenarios including:
  - All routing protocols comparison
  - Large-scale circuit designs
  - VLAN, VTP, and STP configurations
  - Private VLANs
  - High availability protocols (HSRP, VRRP, GLBP)
  - Advanced EtherChannel configurations

### Microsoft Certified Solutions Associate (MCSA)

Windows Server system administration and Active Directory implementation:

- **Final Project**: [Comprehensive Windows Server Project](MCSA/Final%20Project/MCSA_FinalProject.pdf)
  - Multi-domain Active Directory forest (ITI.LOCAL with child domains)
  - Read-Only Domain Controller (RODC) implementation
  - Group Policy Objects (GPO) for user restrictions and software deployment
  - Windows Deployment Services (WDS) for automated OS installation
  - DNS, DHCP, IIS/FTP server configuration
  - Administrative delegation and roaming profiles
  - See [Project Details](MCSA/Final%20Project/README.md)
- **Tasks**: [Hands-on lab exercises](MCSA/Tasks/MCSA_TASKS.pdf)
  - Windows Server administration tasks
  - Active Directory configuration exercises
  - Server role implementations
- **Notes**: Lecture summaries covering:
  - Lectures 1-3: Text format notes
  - Lectures 4-6: Markdown format notes on advanced topics

### Red Hat System Administration (RHSA)

Comprehensive Linux system administration training in two levels:

#### RHSA I
- **Labs**: Hands-on tasks covering fundamental Linux administration
  - Task 1-4: Various system administration exercises
  - Screenshots documenting lab completion
- **Certification**: Official Red Hat training certificate

#### RHSA II
- **Lecture Summaries**: Notes from 6 lectures covering advanced topics:
  - Process management and forking
  - Priority and scheduling
  - User management
  - System monitoring
  - And more...
- **Tasks**: Advanced lab assignments (PDF format)

### VMware vSphere

VMware vSphere server virtualization training covering both theoretical foundations and practical implementation:

- **Lecture Notes** (7 lectures): Comprehensive notes covering:
  - Lecture 1: Hypervisor types (Type 1 & Type 2), VMware ESXi, vSphere components (ESXi + vCenter), SDDC concepts, vMotion, HA, DRS, and storage fundamentals (SAN/NAS)
  - Lecture 2: ESXi resource allocation, VMware Tools, virtual networking (Virtual Standard Switch), file systems (VMFS/NFS/vSAN), ESXi security features, and lockdown mode
  - Lecture 3: vCenter deployment and configuration, Single Sign-On (SSO), Enhanced Linked Mode, HOSTD/VPXD/VPXA services
  - Lecture 4: Active Directory integration, syslog configuration, virtual networking hierarchy, Standard vs. Distributed switches, VLANs, storage (SAN/NAS/vSAN), VMFS datastores, NFS datastores
  - Lecture 5: VM creation from templates, content libraries (local/shared/subscribed), PCI passthrough, EVC (Enhanced vMotion Compatibility), DRS (Distributed Resource Scheduler)
  - Lecture 6: vMotion migration, Fault Tolerance, High Availability, storage migration, and advanced DRS policies
  - Lecture 7: Advanced topics and review
- **Final Project**: [VMware vSphere Infrastructure Design Project](Vmware%20Vsphere/Project/OJA.pdf)
  - Comprehensive end-to-end vSphere environment design and implementation
  - See [Project Details](Vmware%20Vsphere/Project/README.md)
