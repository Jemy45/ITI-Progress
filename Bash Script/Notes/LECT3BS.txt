SED Explanation:

sed " command " 

command is divided into 2 parts 
address: pattern, line number, range, regex
action: Search & replace(Substitute) (s), Print(p), Delete(d), Append(a), Insert(i), Replace line(c), Numbering lines (count)(=)

sed -option " command "

-i    Edit the file in-place (overwrite)
-n    Suppress automatic printing of lines.
-e    Allows multiple commands.
-f    Reads sed commands from a file.
-r    Use extended regular expressions.

eg::
Hna msh bey3dl 3la el file hna bs bye2ra w yetba3 el ta8yer 
sed 's/unix/windows' file.txt => kda hwa hymshy 3la kol satr hy8r awl unix bs b windows 

c => replace all unix with windows

sed 's/unix/windows/2' file.txt => estabdel el tanya bs 

sed 's/unix/windows/2g' file.txt

34an el ta3del ysm3 fe file b2a 
1) using redirect sed 's/unix/windows' file.txt > file2.txt
2) use option -i to overwrite  sed -i 's/unix/windows' file.txt

To ignore case: add option /i if you used /ig then ignore and replace all occurrence of unix
sed 's/unix/windows/i' file.txt

To choose specific lines 

sed '1,3    s/unix/windows' file.txt => kda mn el satr el awl lel talet bs ely ybos fehom
