# CCNA Final Project

## 📁 Folder Contents

This folder contains the Cisco CCNA final project deliverables:

- **[`CCNA Final Project.pkt`](CCNA%20Final%20Project.pkt)**: Cisco Packet Tracer simulation file.
- **[`CCNA Final Project Documentation.pdf`](CCNA%20Final%20Project%20Documentation.pdf)**: full project report with design, configuration, and verification.

## 🌐 Project Overview

This project presents an enterprise-style **software company network architecture** built in Cisco Packet Tracer.

The design focuses on segmentation, scalability, security, centralized services, and high availability for multiple company departments (Software Development, HR, IT, Marketing, Accounting, and Technical Support).

## 🧰 About `CCNA Final Project.pkt`

The `.pkt` file contains the full simulated topology and device configurations, including:

- Department-based VLAN segmentation
- Inter-VLAN routing (Router-on-a-Stick)
- Dynamic routing with OSPFv2 (Area 0)
- DHCP service integration with `ip helper-address`
- EtherChannel (LACP) between access/distribution switches
- HSRP gateway redundancy
- FTP server for configuration backup
- AAA (TACACS+) and SSH-based remote access
- NAT (Static NAT + PAT overload) for internet connectivity

## 📄 About `CCNA Final Project Documentation.pdf`

The PDF documents the project end-to-end and includes:

- Network objectives and architecture explanation
- IP addressing and VLSM subnetting plan
- VLAN design and access/trunk configuration
- Inter-VLAN routing configuration and verification
- DHCP pools and relay configuration
- EtherChannel design/configuration/verification
- OSPF configuration and route verification
- HSRP active/standby setup and validation
- FTP backup workflow for device configurations
- AAA + SSH authentication setup and tests
- NAT/PAT setup and connectivity verification
- Command listings, screenshots, and verification outputs
